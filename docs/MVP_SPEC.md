## 1. 产品概述

### 1.1 产品定位

在线图片背景抠图工具。用户上传图片，自动移除背景，返回透明 PNG。

### 1.2 核心用户故事

> 作为用户，我想上传一张图片并快速获得去除背景的版本，这样我可以方便地把主体用到其他地方（PPT、海报、电商素材等）。

### 1.3 MVP 范围

**做（In Scope）**
- 图片上传（支持 JPG/PNG/WebP，最大 10MB）
- 调用 Remove.bg API 抠图
- 效果预览（原图 vs 结果）
- 下载透明PNG

**不做（Out of Scope）**
- 用户登录 / 账号体系
- 批量处理
- 历史记录
- 边缘优化 / 羽化参数
- 付费订阅

## 5. 技术方案

### 5.1 架构

```
┌──────────────┐      ┌──────────────────┐      ┌──────────────┐
│   用户浏览器   │ ───→ │  Cloudflare      │ ───→ │  Remove.bg   │
│  (Next.js)    │ ←─── │  Worker (代理)   │ ←─── │   API        │
└──────────────┘      └──────────────────┘      └──────────────┘
```

- **前端**：Next.js（部署在 Cloudflare Pages）
- **后端**：Cloudflare Worker（只做转发，保护 API Key）
- **抠图 API**：Remove.bg（按调用次数收费，免费 50 次/月）

### 5.2 API 设计

**Worker 路由**：`/api/remove`

```
请求：POST multipart/form-data
  - image: 图片文件

响应：image/png (直接返回透明PNG)
  或 400/500 + JSON 错误信息
```

### 5.3 Remove.bg API

- 文档：https://www.remove.bg/api
- 免费额度：50 张/月
- 计费：¥ 0.3/张（按量付费，无套餐）
- 支持语言：Python / JavaScript / Ruby / PHP / Go

### 5.4 前端 Cloudflare Pages 配置

- 构建命令：`npm run build`
- 输出目录：`out`
- Node.js 版本：18.x

### 5.5 环境变量

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `REMOVE_BG_API_KEY` | Remove.bg API Key | xxxxx |
